# **App Name**: EduGate Hub

## Core Features:

- Real-time Classroom Attendance: Capture student presence using integrated NFC readers at classroom entrances and display real-time class attendance status for teachers within the application interface.
- Teacher 'Paragon' Alerts: Provide teachers with smart notifications at the start of lessons, summarizing present/absent students and using an intelligent tool to highlight individuals who are in the building but not in their registered class.
- Anti-Fraud Verification Interface: An intuitive interface allowing teachers to conduct quick visual checks, triggering automatic detection of potential card misuse by cross-referencing actual physical presence with NFC tap data.
- Attendance Reporting & Analytics: Generate detailed daily, weekly, and monthly attendance reports for individual students, classes, or the entire school, accessible to authorized personnel.
- Bakaláři System Integration: Seamless, automated transfer of verified attendance records to the Bakaláři school information system via API for streamlined data management and compliance.
- Security Entry Monitoring: Implement the 'Security Bot' as a tool to monitor entries at school access points, automatically notifying staff of unlogged entries or suspicious access patterns to enhance campus security.

## Style Guidelines:

- Primary color: A vibrant, reliable tech-blue (#2289DB) for interface elements that convey smartness and authority in an educational setting. This hue inspires trust and modern efficiency.
- Background color: A very light, clean blue-grey (#ECF4FA) providing a spacious and clear backdrop, evoking an open, professional environment suitable for data display.
- Accent color: A contrasting, energetic indigo-violet (#4D62ED) used to draw attention to critical actions, alerts, or important data points, enhancing user engagement and clarity.
- Headlines and body text will use 'Inter', a clean sans-serif font, chosen for its excellent readability, modern aesthetic, and versatility across various screen sizes and content types in a data-rich application.
- Utilize modern, line-art icons that are clear and universally recognizable for attendance actions, security alerts, and system configurations. Emphasize simplicity and immediate comprehension.
- Employ a grid-based, responsive layout with generous white space to ensure clarity and easy navigation. Information hierarchy will be managed through consistent card-based structures and clear section dividers.
- Incorporate subtle, functional animations for feedback on attendance taps, alert notifications, and state changes (e.g., loading data or submitting reports) to enhance perceived responsiveness and user experience.
